Welcome

Welcome to a first Proof of Concept (PoC) of the NHS Value Set Management Tool (VSMT). We aim to grow the VSMT into a full suite of tools that provide a one-stop-shop for all SNOMED CT value sets, including finding existing sets, creating and maintaining sets, sharing new sets with others, and finding like-minded people to collaborate with. 

In this first PoC we are introducing “Set Checks” - 13 separate checks that run on user supplied SNOMED CT value sets to assess and provide feedback on set content. The focus of these checks is for value sets that are used for data entry to, or data extraction from, electronic patient records. The checks indicate possible errors, content that might be missing, and how a set from an earlier UK SNOMED CT release should be changed due to new or changed content in a more recent UK SNOMED CT release.

Thanks for trying out the VSMT! We would love to hear your feedback, and there is a link on each page of the application to send an email to us.

SNOMED Implementation Team, NHS England

Using the Set Checks Application

To load your value set file, and start using the tool, just click the big green button.

You should have received a Welcome Pack, that contains further information about using the tool. If you haven’t received the pack, or want a new copy, then please contact us via the Feedback link.

By using this tool, you are accepting of the T&Cs. You can re-read the T&Cs at any time using the link provided on each page of the application.
